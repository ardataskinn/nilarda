# 14 Şubat Özel Web Sitesi ❤️

## Kurulum

1. **Görseli ekleyin**: Pembe arka plan üzerindeki kalp görselini `background.jpg` olarak bu klasöre ekleyin.

2. **Müzik dosyasını ekleyin**: `yeniden.mp3` dosyasını bu klasöre ekleyin.

3. **Tarayıcıda açın**: `index.html` dosyasını tarayıcıda açın.

## Özellikler

- ✅ Arka planda otomatik müzik çalma
- ✅ Büyüyüp küçülen animasyonlu başlık
- ✅ Kimlik doğrulama sorusu
- ✅ Doğru cevap kontrolü (cevap: "kedi")
- ✅ Hata mesajı gösterimi
